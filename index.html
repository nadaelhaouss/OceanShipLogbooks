Web map created to visualize the OceanShipLogbooks dataset with interactive panel using leaflet. Available on GitHub.

Features
  • Interactive map
  • Data vizualisation of ship routes from 1750-1850
  • Filtering options

This dataset was acquired from Kaggle, subsequently analyzed, and processed using Python. The Python script filtered out any abnormal data and organized the dataset's structure.

The route data was derived from point data through Python, establishing connections between points belonging to the same trip by leveraging the recorded dates associated with each point.

Marine boundaries were acquired from Marine Regions
